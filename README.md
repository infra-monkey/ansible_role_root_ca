# Overview
This role deploys custom root certificate authorities on the target host.
This role requires root permissions. It must be called as root. This needs to be managed at the ansible or playbook level.

# Variables

| Name  | Type | Required | Default Value | Description |
| ----- | ---- | -------- | ------------- | ----------- |
| custom_ca | list | yes | n.a. | A list of CA to deploy |
| custom_ca.name | string | yes | n.a. | The name of the CA to deploy. Used to build the CA filename |
| custom_ca.content | string | yes | n.a. | The certificate itself |
